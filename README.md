# Stationary-vue

A Vue.js-based frontend for managing stationary items.

## Features

- Vue 3 + TypeScript
- Strict linting and type checking
- Modular structure

## Project Setup

```bash
npm install
```

### Compile and Hot-Reload for Development

```bash
npm run dev
```

### Type-Check, Compile and Minify for Production

```bash
npm run build
```

### Lint with ESLint

```bash
npm run lint
```

## Directory Structure

- `src/` - Source code
- `tsconfig.app.json` - TypeScript configuration
